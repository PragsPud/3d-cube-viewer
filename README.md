# 3D Cube Image Viewer

An interactive web application that allows users to upload 6 images and visualise them as a 3D rotatable cube. Built with Three.js.

![Three.js](https://img.shields.io/badge/Three.js-r128-green.svg)

- **Image Upload Interface**: Upload 6 different images for each face of the cube

- **Interactive 3D Controls**:
  - Left-click + drag to rotate the cube
  - Right-click + drag to pan the view
  - Scroll to zoom in/out
- **Mobile Support**: Touch controls for rotating on mobile devices
- **Responsive Design**: Works seamlessly across desktop and mobile devices
- **Modern UI**: Clean, gradient-based interface with smooth animations
